## 🚀 START THE APP (3 Steps)

Step 1: cd grocery-app
Step 2: chmod +x setup.sh && ./setup.sh
Step 3: Open two terminals

Terminal 1️⃣  (Backend):
  cd backend && source venv/bin/activate && python app.py

Terminal 2️⃣  (Frontend):
  cd frontend && npm start

✅ App opens at http://localhost:3000

## 🚨 TROUBLE? Try This

Port already used?
  lsof -i :5000
  kill -9 <PID>

venv not working?
  source venv/bin/activate  (Linux/Mac)
  venv\Scripts\activate     (Windows)

npm error?
  rm -rf node_modules
  npm install

Database corrupt?
  rm backend/grocery.db
  python init_db.py

## 🆘 EMERGENCY FIXES

Can't start backend?
  → Check Python 3.8+
  → Check port 5000 free
  → Reinstall: pip install -r requirements.txt

Can't start frontend?
  → Check Node 14+
  → Check port 3000 free
  → Reinstall: npm install

API not working?
  → Backend running?
  → Check /api/health endpoint
  → Check CORS in app.py

Database issues?
  → Delete grocery.db
  → Run python init_db.py
