What I learned about Git / GitHub
- Create a new folder and file locally.
- Stage files: `git add <files>`
- Commit changes: `git commit -m "message"`
- Push to GitHub: `git push origin <branch>`

Example (from repository root):

```bash
git add Day4/index.html Day4/README.md
git commit -m "Add Day4 demo and notes about Git usage"
git push origin main
```

Notes:
- Replace `main` with your branch name if different.
- If this is the first push from this machine, you may need to set remote: `git remote add origin <repo-url>` and authenticate.
