# Day 6 - Student Dashboard Frontend Extension

## Overview
A simple frontend extension for an existing student dashboard. This project demonstrates building an interactive dashboard for students to view and manage their academic information.

## Features
- 📚 **Course Management** - View enrolled courses and grades
- 👤 **Student Profile** - Display student information and statistics
- 📊 **Performance Analytics** - Visual representation of academic performance
- 📝 **Assignment Tracker** - Track pending and completed assignments
- 🎯 **Grade Summary** - Quick overview of overall grades

## Project Structure
```
Day6/
├── index.html          # Main dashboard page
├── style.css           # Styling for dashboard
├── app.js              # Dashboard logic and interactions
└── README.md           # This file
```

## Technologies Used
- **HTML5** - Semantic markup
- **CSS3** - Responsive styling with flexbox and grid
- **JavaScript (Vanilla)** - Interactive features without frameworks

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required

### Installation
1. Navigate to the Day6 folder
2. Open `index.html` in your web browser
3. Start exploring the student dashboard

## How to Use
1. **View Dashboard** - Load the page to see your academic overview
2. **Check Courses** - Browse your enrolled courses and grades
3. **Track Assignments** - See pending and due assignments
4. **View Statistics** - Check your performance analytics
